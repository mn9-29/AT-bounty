vorpscript

## How to install

* Copy and paste ```AT-bounty``` folder to ```resources```
* Add ```ensure AT-bounty``` to your ```server.cfg``` file
* Edit ```AT-bounty/config.lua``` to add new coords, peds, weapons, change prices, xp and language
